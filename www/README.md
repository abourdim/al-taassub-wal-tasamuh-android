# 🕊️ Fanaticism & Tolerance — التعصب والتسامح

**بِسْمِ اللَّهِ الرَّحْمَنِ الرَّحِيمِ**

Interactive trilingual (AR/EN/FR) educational web app based on the book **التعصب والتسامح بين المسيحية والإسلام (Fanaticism and Tolerance Between Christianity and Islam)** by Sheikh Mohammed al-Ghazali (1917-1996).

## Features

- 20 tolerance cards with Quranic references, 8 historical tolerance models, daily habit tracker, self-quiz
- Trilingual: Arabic, English, French with RTL/LTR support
- 3 themes (Amethyst, Night, Lavender) with localStorage persistence
- Audiobook narrator with karaoke, voice/speed/pitch, loop, sleep timer
- Responsive design + PWA-ready

## Tech

Pure HTML + CSS + JS. No frameworks. Just open `index.html`.
